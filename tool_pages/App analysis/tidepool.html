About Tidepool

Tidepool is an AI tool developed by Aquarium that specializes in product analytics for AI text interfaces. It employs advanced algorithms to identify patterns in user interactions with text, allowing users to make informed decisions about their products. With Tidepool, users can gain valuable insights into how their product is being used by analyzing user text interactions, without the need for manual event analysis. The tool offers features such as automated insights with embeddings, which clusters similar text together and highlights important attributes in the data. This is achieved with minimal human intervention, making the analysis process convenient and efficient. Tidepool also provides an intuitive visual interface that allows users to explore subcategories of interest and delve into the text of specific conversations, eliminating the need for SQL or programming knowledge.Tidepool enables users to categorize new user interactions based on defined attributes, track trends in user activity, and uncover shifting user behavior over time. By correlating text attributes to product metrics, users can make data-driven product decisions. Moreover, Tidepool is designed to seamlessly integrate with users' existing data infrastructure, whether it be an SDK, CDP, or reverse-ETL.Overall, Tidepool by Aquarium offers a comprehensive solution for AI-based text interface analysis, providing valuable insights and facilitating better product decision-making through its automated analytics capabilities.
