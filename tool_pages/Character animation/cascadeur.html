About Cascadeur

Cascadeur is a standalone 3D keyframe animation software designed for humanoid and other character animations. This tool aims to simplify the animation process, from creating a new animation sequence to editing an existing one. With its artificial intelligence and physics tools, Cascadeur allows users to speed up their work and achieve high-quality outcomes. Features include Quick Rigging for easily generating or editing character rigs, Animation Unbaking for converting baked animations into editable ones, and AutoPosing, an intelligent rigging feature powered by neural networks. Additionally, it offers controllers for detailed hand and finger animations. The AutoPhysics tool allows you to create realistic, natural motion with minimal adjustments. Another feature is the provision of secondary motion, which adds a natural feel to animations through body part overlaps and bounces. Cascadeur is compatible with a variety of file formats including .FBX, .DAE, and .USD, making it an adaptable tool for any animation workflow. Furthermore, the animation software is compatible with a range of other 3D software and platforms such as Unreal, Unity, Daz3d, 3DS Max, Maya, Mixamo, Blender, Houdini, and Cinema 4D.
