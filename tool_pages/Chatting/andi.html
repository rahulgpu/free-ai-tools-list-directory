About Andi

Andi is a novel search tool designed with generative AI technology. Its primary distinction from traditional search engines lies in its response mode - rather than presenting the user with a list of linked sources, Andi provides direct answers to queries. The user experience is modelled to resemble a conversation with an intelligent friend, delivering insightful responses that aim to directly satisfy a user's information need. This reshapes the search paradigm from searching for documents or pages to seeking answers. Leveraging the advancements in generative AI, Andi can generate unique responses based on user queries, making it more dynamic than traditional keyword-based search tools. As such, it can offer an intuitive, interactive, and user-friendly search experience. Note that the tool requires JavaScript enabled and a modern browser for optimal functioning. Its aim is to spearhead the next generation of search tools using the power of AI, redefining how users interact with online information retrieval systems.
