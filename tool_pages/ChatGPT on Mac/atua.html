About Atua

Atua is a tool designed to simplify the use of ChatGPT by allowing access through a single shortcut key on any Mac app. Atua eliminates the need for context switching thus offering seamless AI assistance. The tool is particularly beneficial for text selection and processing; Users can select text from any window, trigger the shortcut key, and let Atua handle the rest. The application also offers the functionality to customize predefined commands, enabling users to tailor the tool to their needs by creating custom commands and assigning hotkeys. This feature can support a wide range of tasks such as rephrasing text, fixing grammar issues, or expanding content. Atua also allows users to save and review their previous conversations with ChatGPT. To use Atua, users are required to have an OpenAI API key and Mac OSX version 10.12 or higher. Please note that while Atua is currently only available for Mac, plans are in place to eventually support Windows and Linux. No data is stored remotely as all data is saved locally on your device, ensuring user privacy.
