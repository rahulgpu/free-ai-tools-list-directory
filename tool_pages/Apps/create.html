About Create

Create is an AI design and dev tool that lets you make components, sites, tools and apps using English. How it works: - Just type how you want it to work. Create generates the code for your app in real time. Use multiple AI models at once. - Paste images in to go from screenshot to code. - Add more details to your spec to get it just right - Build tools powered by integrations like AI (ChatGPT, GPT Vision, Claude 3, Groq, and more), web scraping, Google Maps, charts, and more. - Make apps that talk to any REST API, including your internal data. - Make components to make your app look exactly how you want and share across your team. - Quickly go back to any previous version with full version history built in - Add your team to build together - Publish your app live, or copy and paste or export the code (React, Tailwind, etc.) Use cases: - Component design: build React components much faster for use in your next project. - Prototyping: turn your next great idea into an interactive prototype in seconds. - Landing pages: spin up simple landing pages to test different ideas - Lead gen tools: make interactive lead gen experiences that use AI - Internal tools: make AI tools that talk to your own data
