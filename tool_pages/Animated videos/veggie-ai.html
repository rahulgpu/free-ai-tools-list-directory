About Veggie AI

Veggie AI is an online tool that uses AI technology to generate controllable videos. It provides a simple, flexible, and user-friendly interface so anyone can create controllable videos without needing prior AI knowledge. Users can easily upload character photos, action videos or input text prompts to generate either character photos or action videos. Veggie AI features four unique modes that offer diverse creation methods. The Mix Mode and Animate Mode enable users to upload a character image and either a video or text motion prompt to generate a video. The Ideate Mode allows users to create a video based on the descriptions provided in the text character and motion prompts. The Stylize Mode provides an option to upload a character photo and input text image and motion prompts to create a video where the character is a mixture of the photo and the image prompt, performing actions rendered from the motion prompt.The platform also supports common image formats such as JPEG and PNG, as well as standard video formats like MP4, and prioritises clear and good quality materials for the best results. While its primary use is for single video generation, users with batch processing needs can reach out to Veggie AIs customer support for specific solutions.
