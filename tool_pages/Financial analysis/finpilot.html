About Finpilot

Finpilot is a tool that uses artificial intelligence to assist financial analysts in their investment research, analysis of companies, report writing, and data visualisation. It sources information from verifiable sources including documents and the web, streamlining financial analysis workflows. With its chat feature, users can communicate with this tool to retrieve financial data efficiently, saving them from time-consuming tasks such as sifting through hundreds of documents. Finpilot allows users to focus more on information analysis by handling repetitive data collection tasks. The tool also functions as a reliable query responder, delivering direct information from related documents and the web. It has the capacity to compile data into organized, Excel-compatible tables. Additionally, Finpilot provides a feature for comparing companies, building comparison sets with key metrics and visualising trends across different companies and time periods. All answers and figures provided by Finpilot are linked to their original source documents for instant verification. The tool also enables financial analysis and faster report writing through its interactive document chat feature. Finpilot is primarily aimed at professional equity analysts. The AI agents underpinning Finpilot, built on foundational GPTs and proprietary models, effectively execute professional equity analyst tasks.
