About Blimey

Blimey is an AI image generator tool that allows users to create images with full control over composition, colors and style. It provides an easy-to-use feature for building a 3D scene which guides image generation. Users without previous 3D experience can effortlessly use this tool for creating consistent scenes, characters, and experimenting with multiple camera angles. Blimey also allows users to generate new 3D models when the pre-made models do not fit their desired scene, either by uploading an image as reference or generating one. Another feature that enhances its user-friendliness is the easy navigation system - similar to gaming with WASD or arrow keys. With Blimey, users can upload images that control the image-generation, change backgrounds, save scenes, and import .obj files. Its AI-based image generation capability improves the efficiency and creative freedom of the content creation process.
