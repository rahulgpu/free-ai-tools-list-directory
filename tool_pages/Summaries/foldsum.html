About FoldSum

FoldSum is a Chrome extension tool that enables text summarization and translation using the ChatGPT model. Its main objective is to enhance reading efficiency by allowing users to summarize selected text on web pages. With FoldSum, users can specify the text they want to summarize and replace it directly within the webpage. It supports summarization and translation in multiple languages in a single step. The tool utilizes a web-based ChatGPT or POE engine for summarization and translation, without incurring any additional token cost.To use FoldSum, users need to download and install the browser extension. Once installed, they can open a logged-in ChatGPT or POE page and then switch to another page to select the text they want to summarize. By clicking the "Summarize" button, the selected text will be replaced with the generated summary. FoldSum is free to use and does not impose any extra token cost for summarization and translation.Some additional features of FoldSum include support for Chrome, Brave, Arc, and other Chrome-based browsers, the ability to turn off the floating summary button in the selection area, language customization for translation, and support for nested summaries. FoldSum does not collect user information and does not interfere with the normal use of ChatGPT or POE pages.
