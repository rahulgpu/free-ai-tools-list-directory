About AI Comment Creator

The AI Comment Creator is a WordPress plugin that utilizes GPT-3 technology to generate comments for WordPress posts. The plugin is linked to two third-party API services – RandomUser.me and AIContentGenerator – to produce random data, including names, surnames, and avatars, and to create content using artificial intelligence. By installing the plugin, users allow these API connections. The AI Comment Creator allows users to configure plugin options, including choosing between the OpenAI and AIContentGenerator APIs, setting categories, maximum tokens, language, and prompts for the generated comments. Users can also schedule the automatic comments to be generated at specific times using WP-cron. Additionally, users can choose to add avatar images to the comments. The plugin provides a user-friendly interface that allows users to preview and make changes to the generated comments before they are posted. Although the plugin is free to use, users who want to use the AIContentGenerator API must purchase credits. The AI Comment Creator is aimed at automatically producing unique, creative and SEO-friendly comments on WordPress posts.
