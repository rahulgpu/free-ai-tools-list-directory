About Seeing AI

Seeing AI is a productivity application developed by Microsoft Corporation that serve as a visual aid for the blind and low vision community. The app narrates the world around the user using artificial intelligence technology. The variety of tools it provides include: Short Text, which speaks any text that appears in front of the camera; Documents, a feature that scans and recognizes printed text with its original formatting; Products functionality that scans barcodes and provides information about the products; Scenes, an overall description of the scene captured; People, which recognizes and saves peoples faces; Currency, which identifies different currency notes; Colors, for color identification; and Handwriting, that reads handwritten text. The app further provides a feature to recognize the light level in the surrounding by generating an audible tone. Users can share images from other apps to 'Recognize with Seeing AI' for image description. This tool constantly evolves based on the user community feedback and advances in AI research.
