About Langame card game

Lang Game is an AI-generated conversation card game designed to be enjoyed with friends and family. Using a swiping mechanism, users can select which cards should be part of their personalized deck. These cards are grouped into different categories such as Ice Breaker, Philosophy, Deep Talk, Education, Self Exploration, Relationship, Entrepreneurship, Science, and Startup. These cards provide topics and conversation starters to be used during game play. The game is available for pre-order for $25 and comes with a set of 20 cards. Users can also add their own questions to the deck. The game helps to facilitate meaningful conversations and encourages users to explore new topics.
