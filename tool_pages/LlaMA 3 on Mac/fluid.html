About Fluid

Fluid is a private assistant designed for Mac by Meta Llama 3, offering the functionality of AI tools like ChatGPT whilst focusing on offline usage and unparalleled privacy. Engineered primarily for Macs with Apple Silicon and macOS 14 or later, the Fluid AI assistant operates on users' Macs, ensuring all interactions and communications remain within the device without necessitating a connection to external servers or third-party entities. This independent operation helps secure users data privacy. Fluid's voice control feature allows users to dictate inputs, while assuring that the voice data stays within the device. With a straightforward installation process, no specific skills are needed to start using Fluid. Once installed, launching the assistant is as simple as pressing Ctrl+Space. Fluid prides itself on its security, having been validated by Apple, running in a secure app sandbox without needing admin password and offering automatic updates. For control, users can easily interrupt or stop Fluids operations as needed. In terms of future development, the team plans to incorporate features such as web search capabilities, context awareness, and data memorization for a more personalized user experience.
