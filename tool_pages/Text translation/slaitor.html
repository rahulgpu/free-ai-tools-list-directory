About slAItor

slAItor is an AI-powered translation engine that incorporates recent advances in artificial intelligence to not only translate text but also explain translations and correct errors. It features a style calibrator that adapts translations to different contexts and provides translation alternatives. Users can use the step-by-step feature to learn or improve their understanding of a language and the checker mode to provide their translations, and slAItor will spot, explain, and correct errors. The system is based on GPT AI technology from OpenAI and combines it with state-of-the-art translation technologies. The tool is designed to be easy to use on most devices and has a fully responsive user interface and an easy copy feature. Through the subscription-based service, users can translate files in xslx, docx, txt, and tmx formats and have their translations saved in their account and searchable. The tool is currently available for French, English, Chinese, and Japanese, with support for Dutch, Arabic, Korean, and Hindi coming in the future. Improvement in the speed of long translations is expected towards the end of June, and users with a "Pro" license will have access to new features as soon as they are released.
