About editGPT

editGPT is a powerful AI tool that allows users to proofread, edit, and track changes to their content in chatGPT. It is available as a Chrome and Firefox extension, making it easy to use with the chatGPT platform. editGPT offers a range of features, including an example prompts section for users to get acquainted with the platform, as well as a demo version for users to try out the tool before committing to it. The tool also provides a contact form for users to reach out to the developer, @shuafeiwang, with any queries or suggestions. Overall, editGPT is a reliable and user-friendly AI tool for proofreading, editing and tracking changes to content in chatGPT.
