About IntelliBar

IntelliBar is an AI-powered assistant designed for use on Mac OS. The tool is a ChatGPT client that helps users efficiently perform tasks or answer queries directly on their Mac. It offers a selection-editing feature, enabling the user to modify any visible text, ranging from amending emails to summarizing online articles. This includes changing the tone of a text, fixing grammar, translation, and even refactoring code. IntelliBar is optimized for keyboard use, with a majority of actions achievable through key commands, providing a greater sense of control and quickness in use. Providing AI-based understanding, IntelliBar can also receive and respond to questions about any text, file, or image. A 'Read' feature is available, allowing the user to select any text in any app for IntelliBar to read aloud. Prompts to IntelliBar are sent directly to the OpenAI API, enhancing the security of the process. Furthermore, it is compatible with GPT-4 and utilises OpenAI's latest DALL-E, TTS, and Whispers models for various tasks. Accessibility to IntelliBar requires an OpenAI account, but there is no necessity for a ChatGPT Plus account. Users must provide an OpenAI API key upon first launch, which facilitates IntelliBar's access to the OpenAI's API. Cost depends on usage, with detailed expenditure tracked on the OpenAI dashboard, and option for setting monthly budget limit.
