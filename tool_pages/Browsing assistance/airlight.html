About Airlight

Airlight is a lightweight browsing tool designed to provide access to frequently used web applications on any screen, simulating the functionality of the spotlight search feature on a Mac. This tool allows users to open or hide the tool interface using 'option + space' keys, eliminating the need to switch between multiple browser tabs or applications. Airlight includes a default search functionality leveraging 'perplexity'. It also facilitates coding by providing seamless access to 'chatgpt' without disrupting the coding process. Airlight is available as a one-time purchase and it offers a 7-day free trial. Although primarily designed to access frequent web applications like chatgpt, perplexity, GitHub, Twitter, among others, it does not fully replace your browser but rather enhances productivity by reducing the need for multiple tabs. Airlight supports MacOS 10.3+ (both Intel and Arm64), maintaining a privacy-first approach, as it does not store any user data on its servers beyond email address and name required for license purposes.
